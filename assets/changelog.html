Changelog

0.8.0 (Dec.XX 2011)

  • Reorder main list activities (New, Albums, Artist, etc.).
  • Internal changes to make SqueezeControl run smoothly on the emulator.

0.7.3 (Nov.11 2010)

  • Fixed a memory leak

  • Added a now playing toolbar with quick access to main screen
  • Added fast scroll to lists
  • No longer runs as a service (don't worry, it was definitely not spying on you, but there was noe real need for it)
  • Fixed download of songs with question marks in their names
  • Better display of current Internet Radio

0.7.2 (Aug.8 2010)

0.7.1 (Aug.7 2010)

  • Fixed fast scrolling bugs in all views

0.7.0 (Aug.7 2010)

  • Experimental apps support
  • Genre bugs fixed
  • Various bug fixes and minor improvements

0.6.3 (Aug.6 2010)
